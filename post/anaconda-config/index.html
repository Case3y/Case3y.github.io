修改 Jupyter Notebook 默认启动路径

输入 jupyter notebook --generate-config 生成 jupyter_notebook_config.py 配置文件

打开配置文件找到以下代码：

1

#c.NotebookApp.notebook_dir = ''

删除注释，然后添加自己想要的路径

1

c.NotebookApp.notebook_dir = '路径'

修改 Jupyter Notebook 默认字体

Jupyter 默认字体是 monospace，在 Windows 下字体为 Courier New，觉得看代码不太方便，于是找到 CSS 文件，路径如下：

1

D:\Anaconda\Lib\site-packages\notebook\static\components\codemirror\lib\codemirror.css

将字体修改为 Consolas：

1

font-family: Consolas, monospace;

jupyter notebook 添加虚拟内核

一、安装 ipykernel

先激活虚拟环境，然后输入 conda install ipykernel 安装 ipykernel

二、添加虚拟环境到 Jupyter Notebook

输入以下代码：

1

python -m ipykernel install --user --name 虚拟环境名称 --display-name "Jupyter Notebook 上显示的名称"

三、查看和删除 Jupyter 环境内核

查看内核：

1

jupyter kernelspec list

删除内核：

1

jupyter kernelspec remove <env-name>
